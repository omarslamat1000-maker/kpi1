# الأمان وحدود النسخة

هذه نسخة Static MVP قابلة للنشر على GitHub Pages، وتستخدم `localStorage` للتخزين المحلي داخل المتصفح.

- لا تستخدم كلمات مرور حقيقية أو بيانات حساسة داخل هذه النسخة.
- الحسابات الموجودة حسابات تجريبية محلية وليست مصادقة تشغيلية حقيقية.
- لا توجد قاعدة بيانات مركزية أو صلاحيات خادمية.
- النسخة التشغيلية الحقيقية يجب أن تستخدم Backend مثل Supabase أو Firebase أو API مخصص مع مصادقة حقيقية وسجلات تدقيق مركزية.

للتجربة فقط: كلمة المرور الافتراضية للحسابات التجريبية هي `demo`.
